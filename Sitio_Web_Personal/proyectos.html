Aplicación web para la centralización de datos de aplicaciones de la Administración del Principado de Asturias

Descripción: Desarrollo de una aplicación web para la empresa Inetum, que permite centralizar datos de aplicaciones del Principado de Asturias. Incluye funcionalidades de gestión de aplicaciones, entornos, logs y usuarios de prueba, utilizando Angular en el frontend y Spring en el backend.
